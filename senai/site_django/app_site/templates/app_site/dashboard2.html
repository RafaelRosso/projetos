Dashboard 2

{% csrf_token %}

Este dashboard permite uma visualização gráfica e numérica distribuída por região geográfica, conforme os filtros aplicados, de dados referentes ao quantitativo de inscritos, média geral e média em cada prova. Adicionalmente, é possível uma visualização gráfica em colunas das médias em cada prova conforme a Unidade da Federação e de acordo com as seleções escolhidas nos filtros. Use o cursor do mouse sobre os elementos gráficos para destacar uma informação.

O painel referente ao Local do Exame, no canto inferior direito do dashboard, permite uma visualização do local de exame, UF, média geral e quantidade de inscritos, sendo munido de uma barra de rolagem vertical, ativada ao se posicionar o cursor do mouse sobre o referido painel. Observe que é possível alternar entre ordem crescente ou decrescente com o clique do mouse nos rótulos de colunas onde essa alternância é aplicável.

No canto inferior esquerdo, os mapas revelam dados dos quantitativos de inscritos e média geral por Unidade da Federação ao se posicionar o cursor sobre a representação geográfica de cada ente federativo. Estes dados resultantes também são sensíveis aos filtros aplicados no menu suspenso no topo da página.
